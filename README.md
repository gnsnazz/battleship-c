# Battleship in C (Battle6)

Terminal-based implementation of the Battleship game, developed in C as a group project.

## Features

* Single player mode (Player vs Computer)
* Multiplayer mode (1 vs 1, turn-based)
* Save and load game functionality
* Special attacks in addition to standard shots
* Playable via CLI

## Technologies

* Language: C
* Interface: Command Line (CLI)

## How to Compile

Use `gcc` to compile the project:

```bash
gcc *.c -o battleship
```

## How to Run

```bash
./battleship
```

(On Windows: `battleship.exe`)

## Game Modes

* New Game: Start a fresh match
* Load Game: Resume a previously saved match

## Gameplay

Players take turns attacking opponent coordinates.
In addition to standard attacks, special moves are available depending on the game rules.
